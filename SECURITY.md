# Security Policy

## Reporting a Vulnerability

GitHub Private vulnerability reporting 経由でお願いします:

`https://github.com/<ORG>/cms/security/advisories/new`

（`<ORG>` は本リポジトリの GitHub Organization 名）

## Response SLA

- 受領確認: 7 日以内
- 初期評価: 14 日以内
- 修正: 深刻度に応じて

## Supported Versions

`main` ブランチのみサポート。
